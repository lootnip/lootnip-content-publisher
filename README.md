# Lootnip Content Publisher

Lootnip Content Publisher is a creator-focused publishing tool for preparing and posting short-form content to social media platforms.

The project is designed to help creators manage content publishing workflows across multiple platforms, including TikTok, YouTube, Instagram, Facebook, Threads, and Tistory.

## Current Purpose

- Prepare short-form video posting workflows
- Support creator-controlled publishing
- Organize publishing requests, captions, and platform-specific metadata
- Extend toward automated or semi-automated social content distribution

## TikTok Review Context

For TikTok review purposes, this app is intended to:

- connect an authorized creator account,
- prepare video posting data,
- upload short-form video content, and
- publish content only after the user authorizes the app and requested permissions.
